This is an experiment to learn about APIs as applied to Web pages woithout Wordpress. constructivism and technology in K12 and higher ed. My experience includes private and public institutions and non-profits. My interests are varied and include multimedia creation, data visualization, history, APIs, poetry, WordPress, and photography.

The idea is copied from Tom Woodward http://bionicteaching.com/portfolio-work-interweaving-the-personal-api/. This is a portfolio created with the use of programming APIs.

"This site was built not just to represent me and my work but also to explore how I could use various APIs, Angular, and JavaScript to create an interactive experience where the content is generated through other tools and platforms."
